# Netflix Data Analysis Project

## 📊 Overview
This project analyzes Netflix movies and TV shows dataset using Python.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📈 Features
- Data Cleaning
- Data Visualization
- Outlier Detection
- Pie Charts, Box Plots, Pair Plots

## 📁 Dataset
Netflix Titles Dataset

## 📌 Insights
- Most content is Movies
- Drama is the most common genre
- Content increased after 2015
